## 一、沟通与语言（20问）

### Q001：孩子2岁了还不会说话，正常吗？

**家长困惑**：
我的孩子2岁了还不会说话，只会"啊啊啊"，这正常吗？是不是发育迟缓？

**专业解答**：
每个孩子发展速度不同，但2岁是完全不说话需要关注的年龄。

语言发展的参考：
- 12个月：会说1-2个词（妈妈、爸爸）
- 18个月：会说10-15个词
- 24个月：会说50个词，能说短句
- 3岁：会说简单句子，能被陌生人理解

**需要注意的信号**：
- 12个月对名字没反应
- 18个月不会用手势（指、挥手）
- 2岁不会说任何词
- 2岁后语言能力退步

**实用建议**：
1. 先观察其他发展里程碑
2. 记录孩子是否会用其他方式沟通（手势、表情）
3. 和儿科医生讨论，看是否需要语言评估
4. 如果有担忧，早评估早干预

**关键要点**：
```
✓ 2岁完全不说话需要引起重视
✓ 观察孩子是否有其他沟通方式（手势、表情）
✓ 不会说话≠不想说话
✓ 尽早评估，明确孩子情况
```

**相关概念**：功能性沟通训练、强化、动机操作

### Q002：孩子有语言但不用来沟通

**家长困惑**：
孩子会说话，但都是自言自语或者重复动画片台词，不用来和我沟通。这是为什么？

**专业解答**：
这在自闭症孩子中很常见，叫做"鹦鹉学舌"或"延迟仿说"。语言被用来满足感官需求或自我娱乐，而不是作为沟通工具。

**原因分析**：
- 语言可能是自我刺激（听觉刺激）
- 缺乏功能性沟通训练
- 没有学会语言是交换信息的工具

**实用建议**：
1. 评估孩子是否有功能性语言
2. 创造需要沟通的情境（把喜欢的东西放高）
3. 教替代沟通方式（手势、图片）
4. 强化真正的沟通行为
5. 减少自言自语的机会（换环境）

**关键要点**：
```
✓ 自言自语≠功能性语言
✓ 需要教语言是沟通工具
✓ 创造需要沟通的情境
✓ 强化用语言满足需求
```

**相关概念**：功能性沟通训练、泛化、强化

### Q003：怎么教孩子说话？

**家长困惑**：
我想教孩子说话，但不知道从哪里开始。应该先教什么？

**专业解答**：
教语言有顺序，通常从表达需求开始。

**建议的教学顺序**：

第一阶段：表达需求
1. 用手势表达（指、伸手）
2. 仿说单个词
3. 用词表达需求（"要"、"苹果"）
4. 组合词（"我要苹果"）
5. 问句（"这是什么？"）

**实用建议**：
1. 先教孩子最需要的
2. 从简单到复杂
3. 每次只教一个目标
4. 创造需要语言的情境
5. 立即强化正确的表达

**关键要点**：
```
✓ 从表达需求开始
✓ 先教孩子最想要的
✓ 创造沟通机会
✓ 立即强化
```

**相关概念**：任务分解、强化、塑造、辅助

### Q004：孩子不说新词怎么办

**家长困惑**：
教了很多遍，孩子就是不学新词。每次教他，他都不理我。

**专业解答**：
孩子不学可能有几个原因。

**可能的原因**：
1. 动机不够（强化物不够吸引）
2. 任务太难（超出能力范围）
3. 教学方式不对
4. 孩子不理解要求
5. 强化不及时

**实用建议**：
1. 找到更有效的强化物
2. 降低任务难度
3. 变换教学方式（示范、手势辅助）
4. 确保孩子注意你
5. 孩子一开口就给强化

**关键要点**：
```
✓ 不是孩子不愿意学，是方法不对
✓ 先找到孩子的强化物
✓ 从简单的开始
✓ 确保孩子注意力在你这
```

**相关概念**：强化物、辅助、动机操作

### Q006：怎么扩大孩子的词汇量？

孩子只会说几个词，怎么让他说更多？

**专业解答**：
扩大词汇需要大量暴露和强化。

**实用建议**：
1. 日常生活中多说
2. 每次接触新东西都说名称
3. 描述孩子在看、在玩的东西
4. 反复重复
5. 在不同情境中重复同一词

```
类别1：名词（东西的名称）
→ 苹果、香蕉、球、车

类别2：动词（动作）
→ 吃、喝、跳、跑

类别3：形容词（描述）
→ 大、小、红、绿

类别4：社交用语
→ 你好、再见、谢谢
```

**关键要点**：
```
✓ 日常生活中大量输入
✓ 描述孩子感兴趣的
✓ 在不同情境重复
✓ 立即强化
```

4. 回答后要求孩子仿说或指认
5. 强化等待和正确的回应

**关键要点**：
```
✓ 这是仿说不是功能性提问
✓ 教等待和目光接触
✓ 抓住手让他注意你
✓ 强化正确的回应
```

**相关概念**：刺激控制、辅助、强化

### Q008：孩子不会回答问题

**家长困惑**：
我问孩子问题，他不回答。他知道答案，但就是不说。

**专业解答**：
不会回答问题可能是缺乏训练。

**实用建议**：
1. 从简单的、孩子知道答案的问句开始
2. 先用选择题（"要苹果还是香蕉？"）
3. 提示孩子回答
4. 立即强化正确的回答
5. 逐渐增加问题难度
6. 练习不同类型的问题

**问题难度顺序**：
```
第一级：命名（这是什么？）
第二级：选择（你要苹果还是香蕉？）
第三级：肯定/否定（这是苹果吗？）
第四级：谁/哪里（谁有球？球在哪里？）
第五级：为什么（为什么天是蓝的？）
```

**关键要点**：
```
✓ 从简单开始
✓ 用辅助帮助成功
✓ 立即强化
✓ 逐步增加难度
```

### Q011：孩子理解有问题还是表达有问题？

**家长困惑**：
孩子有时候听不懂我说话，但有时候又能听懂。不知道他是听不懂还是不想理我。

**专业解答**：
理解（接收语言）和表达（发出语言）是不同的能力。

**简单测试**：
1. 用手势配合指令（"指一指球"）
2. 如果能理解手势，可能理解力可以
3. 调整语言复杂度

**实用建议**：
1. 先确定孩子理解水平
2. 简化语言
3. 配合视觉提示
4. 确保孩子注意你
5. 记录孩子能理解什么

**理解vs表达**：
```
孩子能理解 > 孩子能表达：正常，早期表达发展慢于理解
孩子能表达 > 孩子能理解：需要评估理解力
```

**关键要点**：
```
✓ 理解力和表达力需要分别评估
✓ 先确定理解水平
✓ 简化语言
✓ 配合视觉提示
```

**相关概念**：刺激控制、辅助

### Q012：孩子不会提要求

```
第一层：完全辅助（握着孩子手指向）

第三层：示范（你先示范说"我要XX"）
第四层：言语提示（你说"我要XX"）
第五层：独立
```

**关键要点**：
```
✓ 创造需要沟通的情境
✓ 把强化物放在能看到但拿不到的地方

✓ 逐渐减少提示

### Q013：孩子鹦鹉学舌怎么办

孩子总是重复我说的话，我说"你好"，他也说"你好"。这是什么问题？

**类型**：
1. 即时仿说：重复刚听到的话
2. 延迟仿说：重复之前听过的话

**可能的原因**：
- 学习说话的自然过程
- 感官自我刺激
- 不知道如何回应
- 刻板行为

**实用建议**：
1. 判断是即时还是延迟仿说
2. 评估功能（是沟通还是自我刺激）
3. 教替代的回应方式
4. 减少对仿说的强化
5. 创造需要功能性语言的情境

**关键要点**：
```
✓ 仿说可能是学习过程
✓ 也可能是自我刺激
✓ 教功能性回应
✓ 减少对仿说的强化
```

**相关概念**：消退、替代行为、功能性沟通训练

### Q018：怎么教孩子问"为什么"

**家长困惑**：
孩子会回答"为什么"的问题，但从来不主动问"为什么"。怎么教他主动问？

**专业解答**：
教孩子主动问问题需要创造机会。

**教学策略**：
1. 当孩子问"这是什么"时，先不回答
2. 示范"为什么"
3. 问"为什么"
4. 回答后问"为什么"
5. 奖励主动问"为什么"

**具体步骤**：
```
第一步：建立模式
→ 孩子问"这是什么？"
→ 回答后反问"你知道为什么吗？"

惩罚可能短期有效，但有很多问题。

**惩罚的短期效果**：
```
✓ 孩子立即停止行为
✓ 家长感觉"管教有效"
✓ 表面上看问题解决了
```

**惩罚的长期问题**：
```
✗ 不教替代的正确行为
✗ 可能增加焦虑和恐惧
✗ 可能影响亲子关系
✗ 可能学会用暴力解决问题
✗ 惩罚减轻后会反弹
✗ 只在惩罚在场时有效
```

**什么时候可以考虑温和的惩罚**：
1. 作为最后手段
2. 在专业指导下
3. 配合强化替代行为
4. 惩罚要温和、短暂
5. 不能有身体或心理伤害

**建议**：
```
首选：强化正确行为
次选：消退问题行为
最后：温和的惩罚（仅在专业指导下）
```

**关键要点**：
```
✓ 惩罚短期有效但长期有害
✓ 首选强化正确行为
✓ 消退配合强化替代行为
✓ 惩罚需要专业指导

**相关概念**：强化、惩罚、替代行为
