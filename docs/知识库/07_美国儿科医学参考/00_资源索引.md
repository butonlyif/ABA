# 美国儿科医学权威资源索引

> 版本：v0.1
> 创建日期：2026-05-17
> 用途：为现有 ABA 知识库（01–06）补充更宽泛的美国儿科医学背景，便于系统学习与查证
> 维护建议：每季度复核一次链接与版本号；新增资料统一归到本目录的子文件夹

---

## 使用说明

本索引按"权重 + 用途"分组，每条目包含：**名称**、**类别**、**访问方式**、**简介**、**链接**。

- 🟢 = 完全免费 / 开放获取
- 🟡 = 部分免费（摘要免费、全文需订阅 / 注册）
- 🔴 = 需付费购买或机构订阅

学习路径建议：
1. **先建框架**：用 Nelson 教材或 StatPearls 系统过一遍儿科知识体系
2. **再补指南**：用 AAP Clinical Practice Guidelines + Bright Futures 掌握"最新做法"
3. **持续追踪**：用 PubMed / AAP 期刊订阅追踪本领域更新
4. **场景查证**：遇到具体问题时，从 MedlinePlus / HealthyChildren.org 查家长向解释，从 UpToDate / StatPearls 查临床版

## 一、权威教材（系统学习的主轴）

### 1.1 Nelson Textbook of Pediatrics（22nd Edition, 2024–2025）🔴

- **类别**：综合儿科教材，全球被引最广
- **出版方**：Elsevier；主编 Robert M. Kliegman、Joseph W. St. Geme III
- **特点**：22 版是最新版（前一版 2020），新增大量图表、视频，并提供在线持续更新
- **访问方式**：购买纸质 / 电子书；机构订阅 ClinicalKey 可在线检索全文
- **官方页面**：<https://shop.elsevier.com/books/nelson-textbook-of-pediatrics-2-volume-set/kliegman/978-0-323-88305-4>
- **ClinicalKey**：<https://www.clinicalkey.com/#!/browse/book/3-s2.0-C2020103101X>

### 1.2 Rudolph's Pediatrics 🔴

- **类别**：另一部主流儿科综合教材，临床导向
- **出版方**：McGraw-Hill (AccessPediatrics)
- **访问方式**：机构订阅 AccessPediatrics 可在线阅读

### 1.3 Zitelli and Davis' Atlas of Pediatric Physical Diagnosis 🔴

- **类别**：图谱型，体征识别用
- **出版方**：Elsevier
- **适合**：补充体征图片，识别皮肤、口腔、面部异常等

### 1.4 Pediatric Clinical Practice Guidelines & Policies（AAP，25/26 版）🔴

- **类别**：把 AAP 所有政策、临床报告、技术报告汇编成一本工具书
- **出版方**：American Academy of Pediatrics
- **官方购买**：<https://www.aap.org/Pediatric-Clinical-Practice-Guidelines-Policies-A-Compendium-of-Evidence-based-Research-for-Pedia>

## 二、临床指南与政策（"最新标准"的来源）

### 2.1 AAP Clinical Practice Guidelines（开放阅览）🟢

- **类别**：美国儿科学会临床实践指南合集
- **特点**：自 2025 年起所有新指南采用 **GRADE** 方法学
- **重点指南**（近年发布）：
  - 儿童青少年肥胖评估与治疗
  - ADHD 诊断、评估与治疗
  - 高血压筛查与管理
  - 新生儿高胆红素血症管理（≥35 周）
  - 门诊急性疼痛阿片处方
- **入口**：<https://publications.aap.org/collection/523/Clinical-Practice-Guidelines>
- **AAP 官方版块**：<https://www.aap.org/en/quality-improvement/clinical-practice-guidelines/>

### 2.2 Bright Futures Guidelines, 4th Edition（AAP）🟡

- **类别**：儿童/青少年预防保健 + 健康监督的"圣经"
- **覆盖**：从婴儿到青少年每次健康儿童访视应做什么（筛查、检查、免疫、指导）
- **新增重点**：健康社会决定因素、终身身心健康、母亲抑郁筛查、安全睡眠、母乳喂养婴儿铁补充、氟漆、血脂筛查
- **关键工具**（部分免费 PDF）：
  - Periodicity Schedule（推荐访视时点图）：<https://downloads.aap.org/AAP/PDF/periodicity_schedule.pdf>
  - Pocket Guide（4 版口袋手册 PDF）：<https://fhop.ucsf.edu/sites/g/files/tkssra4481/f/Bright_Futures_4th_Edition_POCKETGUIDE.pdf>
  - 主页：<https://www.aap.org/en/practice-management/bright-futures/bright-futures-materials-and-tools/bright-futures-guidelines-and-pocket-guide/>

### 2.3 CDC 儿科核心资源 🟢

- **儿童/青少年免疫接种程序**：<https://www.cdc.gov/vaccines/hcp/imz-schedules/child-adolescent-age.html>
  - ⚠️ **2026 年 1 月重大更新**：CDC 重构了儿童免疫程序，采用"三层分类"（普遍推荐 / 高风险人群 / 共同临床决策），并将"普遍推荐"疫苗预防的疾病数从 18 减到 11；HPV 从两剂改为一剂。引用时务必区分 2025 vs 2026 版本。新闻稿：<https://www.cdc.gov/media/releases/2026/2026-cdc-acts-on-presidential-memorandum-to-update-childhood-immunization-schedule.html>
- **追赶免疫程序**：<https://www.cdc.gov/vaccines/hcp/imz-schedules/child-adolescent-catch-up.html>
- **发育里程碑（家长版）**：<https://www.cdc.gov/parents/infants/milestones-and-schedules.html>
- **Learn the Signs. Act Early.（早期识别项目）**：<https://www.cdc.gov/act-early/resources/growth-chart.html>
- **生长曲线（Growth Charts）**：CDC 0–2 岁推荐使用 WHO 标准，2–20 岁用 CDC 标准
- **自闭症（ASD）治疗与干预**：<https://www.cdc.gov/autism/treatment/index.html>

### 2.4 USPSTF Pediatric Recommendations 🟢

- **类别**：美国预防服务工作组关于儿科预防干预的循证推荐
- **链接**：<https://www.uspreventiveservicestaskforce.org/>（按 age + topic 检索）

## 三、开放获取学习平台（强烈推荐免费用）

### 3.1 StatPearls 🟢

- **类别**：同行评审、持续更新的临床文章库，覆盖儿科 + 全部专科
- **特点**：每篇含发病机制、临床表现、诊断、治疗、护理要点；可做 CME
- **入口**：<https://www.statpearls.com/>
- **NCBI Bookshelf 镜像（推荐用这个）**：<https://www.ncbi.nlm.nih.gov/books/NBK430685/>
- **示例条目**：
  - Pediatric Appendicitis：<https://www.ncbi.nlm.nih.gov/books/NBK441864/>
  - Pediatric Fluid Management：<https://www.ncbi.nlm.nih.gov/books/NBK560540/>
  - Pediatric Functional Constipation：<https://www.ncbi.nlm.nih.gov/books/NBK537037/>

### 3.2 NCBI Bookshelf 🟢

- **类别**：NIH 旗下开放教科书、报告、临床综述聚合
- **入口**：<https://www.ncbi.nlm.nih.gov/books/>
- **重点条目**：
  - Evidence Base for Applied Behavior Analysis：<https://www.ncbi.nlm.nih.gov/books/NBK619281/>

### 3.3 OPENPediatrics（哈佛波士顿儿童医院出品）🟢

- **类别**：全球儿科临床医生协作的开放学习平台
- **特点**：视频课程、模拟操作、CME 学分（CME 部分有小额费用）
- **主页**：<https://www.openpediatrics.org/>
- **OPENPediatrics Scholar（同行评审的数字学习资源）**：<https://www.openpediatrics.org/scholar>

### 3.4 PedsCases 🟢

- **类别**：阿尔伯塔大学发起的儿科教育资源（医学生 + 住院医友好）
- **内容**：485+ 集播客、视频、信息图、临床案例
- **主页**：<https://www.pedscases.com/>
- **播客列表**：<https://pedscases.com/podcasts>

### 3.5 MedlinePlus（NIH/NLM）🟢

- **类别**：面向公众的健康百科（家长向解释的权威来源）
- **特点**：每个主题配药物信息、检验项目说明、视频
- **主页**：<https://medlineplus.gov/>

### 3.6 HealthyChildren.org（AAP 家长版）🟢

- **类别**：AAP 直接面向家长的官方科普
- **特点**：从喂养、安全、行为到疾病应对，权威且通俗
- **主页**：<https://www.healthychildren.org/>

## 四、文献检索与循证医学

### 4.1 PubMed 🟢

- **类别**：美国国立医学图书馆文献检索（>3500 万条记录）
- **入口**：<https://pubmed.ncbi.nlm.nih.gov/>
- **常用过滤**：限定 child / adolescent + meta-analysis / RCT / systematic review

### 4.2 Cochrane Library 🟡

- **类别**：系统综述黄金标准
- **入口**：<https://www.cochranelibrary.com/>
- **特点**：摘要免费；全文部分国家有公共许可可免费看

### 4.3 核心期刊

- **Pediatrics**（AAP 旗舰）：<https://publications.aap.org/pediatrics>
- **JAMA Pediatrics**：<https://jamanetwork.com/journals/jamapediatrics>
- **The Journal of Pediatrics**（Elsevier）：<https://www.jpeds.com/>
