# 分步训练项目清单（初级/中级/高级）

本清单来自《应用行为分析(ABA)完整教程》初级/中级/高级技能分步训练三册，按领域和难度整理。
每个项目在「图片卡片」中有配套卡片，可在「任务清单」「训练记录」中按回合式教学(DTT)训练。

## 学业前技能
- **初级**：数量概念、理解字母的初始发音、配对大写字母与小写字母、阅读：将字母发音与相关图片配对、阅读：将文字与图片配对
- **中级**：阅读：为图片配对短语和句子
- **高级**：写作：写作流畅性、写作：写句子、数学：0～12以内的数字乘法、数学：故事题——减法、数学：故事题——加法、数学：熟练性、数学：识别故事题中的关键词、阅读：阅读流畅性

## 情绪调节技能
- **初级**：接受性和表达性语言技能（情感）
- **中级**：接受性和表达性语言技能（辨别或命名复杂的情绪）
- **高级**：系统脱敏疗法：关灯睡觉、系统脱敏疗法：独自待在房间里、系统脱敏疗法：穿带有纽扣的衣服

## 社交技能
- **中级**：考虑自己与考虑他人、适当行为与不当行为、通过提问来获得信息
- **高级**：20个问题游戏、不妥协的谈判与妥协的谈判、令人不快的行为与友善的行为、识别社交语言

## 自理技能
- **中级**：安全行为与危险行为
- **高级**：吃健康食物、将社区标志与描述配对、急救：识别需要急救用品的情况、购物：从自动售货机购买商品、驾驶员考试：辨别交通标志

## 视觉空间技能
- **初级**：哪个不是同类物品、序列模式扩展、按照系列或序列进行排列、接受性和表达性语言技能（三维形状）、根据功能分类物品、根据特征分类图片、部分与整体的关系、配对相关联的物品
- **中级**：常识与推理：图片中的荒谬处、按故事情节排序、按照日常活动顺序排列图片、按照社交场景排列图片、接受性和表达性语言技能（辨别或命名各种材料构成）、相同和不同、美术技能：画画、部分与整体的关系Ⅱ
- **高级**：空间关系、视觉完形、视觉记忆、视觉辨别

## 语言技能
- **初级**：不规则复数、不规则过去时态的动词、回答简单的关于“什么”的问题、回答简单的关于“哪一个”的问题、回答简单的关于“谁”的问题、回答简单的是否问题、复数、扩展句子长度（动作-物品）、扩展句子长度（对象-动作-物品）、接受性和表达性语言技能（反义词）、接受性和表达性语言技能（属性）、接受性和表达性语言技能（性别）、接受性和表达性语言技能（社会服务人员）、接受性和表达性语言技能（钱）、接受性语言技能（代词_他她）、接受性语言技能（代词_他的她的）、描述图片、理解各种物品的功能、理解房间及房间中的各项物品的概念、理解类别的概念、理解身体各部分的功能、表达性语言技能（代词_他她）、表达性语言技能（代词_他的她的）、辨别声音、过去时态的动词
- **中级**：名词描述、大问题与小问题、将来时态、抽象概念：最喜欢的、抽象概念：真实与虚构、接受性和表达性语言技能（根据动作辨别或命名社会服务人员）、接受性语言技能（根据描述识别地点）、接受性语言技能（根据描述识别物品）、接受性语言技能（辨别复杂的类别）
- **高级**：比喻性语言：习语、比喻性语言：明喻
