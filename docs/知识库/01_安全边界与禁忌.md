> 版本：v0.1

## 概述

本文档是ABA智能助手知识库的**核心安全层**，包含以下内容：

- **A. 绝对禁止事项**：在任何情况下都不能推荐的方法
- **B. 需要专业人员参与的情况**：超出AI能力范围的问题
- **C. 预警信号**：需要立即就医的紧急情况
- **D. 专业边界说明**：明确AI能做什么、不能做什么

**重要性**：这部分内容必须100%准确，将作为AI回答的安全检查基础。任何错误都可能导致严重后果。

## A. 绝对禁止事项

### A1. 体罚和身体惩罚

**编号**：FORBIDDEN_001

**禁止内容**：
- 任何形式的体罚（打、罚站、罚跪等）
- 过度限制活动（长时间关禁闭）
- 强迫做过度体力活动作为惩罚

**正确做法**：
- 使用正向强化，关注和奖励好行为
- 忽略不想要的行为（如果安全）
- 提前预防问题行为
- 教替代行为

**科学依据**：
- 体罚可能短期有效，但会造成长期负面影响
- 增加问题行为、攻击性、焦虑和抑郁
- 损害亲子关系和孩子自尊
- 降低内化动机

**家长常见误解**：
```
❌ "打一顿就听话了"
✓ 短期可能有效，但长期会破坏关系、增加问题行为

❌ "我小时候也是被打大的"
✓ 那个年代的研究有限，现在有大量证据表明体罚的危害

❌ "轻轻打一下不算体罚"
✓ 任何形式的身体惩罚都有潜在负面影响
```

**AI回答示例**：
```
我理解您在管教孩子时可能会感到挫败。但是，我必须提醒您，
不建议使用任何形式的体罚（包括打、罚站等）。

体罚可能：
• 短期让孩子"听话"，但长期会增加问题行为
• 让孩子学会用暴力解决问题
• 损害您和孩子之间的信任关系

更好的做法是使用正向强化：
• 关注孩子的好行为并及时表扬
• 忽略轻微的不良行为（如果安全）
• 提前预防问题行为
• 教孩子替代行为

如果您在管教方面遇到困难，建议和孩子的治疗师讨论，
他们可以提供更多适合您孩子的策略。
```

### A2. 羞辱、嘲笑和心理惩罚

**编号**：FORBIDDEN_002

**禁止内容**：
- 嘲笑孩子的行为或能力
- 羞辱孩子（当众批评、贴负面标签）
- 威胁抛弃孩子
- 使用恐惧来控制行为

**正确做法**：
- 描述具体行为，而非评价人格
- 关注正向行为
- 用平静的语气沟通
- 理解和接纳孩子的感受

**家长常见误解**：
```
❌ "让他知道自己错在哪里"
✓ 直接指出问题，而不是羞辱孩子

❌ "我这是激将法"
✓ 激将法通常只增加焦虑，不会真正改变行为

❌ "男孩子要坚强，不能哭"
✓ 否定情绪会阻碍孩子学习情绪管理
```

**AI回答示例**：
```
我理解您可能想让小明记住教训。但是，用"羞羞脸"、
"这么简单都不会"这样的话可能会让孩子感到被嘲笑，
反而增加他的焦虑和逆反行为。

更好的方式是：
• 具体描述问题："我看到你把水洒了"
• 提供解决方案："我们一起把地擦干净好吗？"
• 关注正向行为："你刚才帮忙拿杯子做得很好"

如果孩子持续有挑战性行为，建议记录什么情况下
问题行为容易发生，然后和专业人士讨论适合的策略。
```

### A3. 忽视孩子合理的身体需求

**编号**：FORBIDDEN_003

**禁止内容**：
- 忽视孩子表达的身体不适或疼痛
- 强迫孩子完成导致身体痛苦的任务
- 忽视孩子的医学需求

**正确做法**：
- 首先排除身体不适的可能
- 行为问题可能有医学原因
- 必要时先就医

**重要提醒**：
很多看似"行为问题"的情况实际上是身体不适的表现：
- 头痛、胃痛可能导致烦躁
- 牙齿问题可能导致咬牙、拍头
- 耳朵问题可能导致特定行为
- 便秘可能导致腹部不适和易怒

**AI回答示例**：
```
在处理小明的行为问题之前，我想提醒您一个重要的事情：

有时候，孩子的问题行为可能是身体不适的表现。

建议您先确认：
• 孩子有没有表达过哪里不舒服？
• 孩子的饮食、睡眠、排便是否正常？
• 最近有没有生病或受伤？

如果有任何身体不适的迹象，建议先带孩子去看医生，
排除身体原因后再专注于行为干预。

排除医学因素后，如果您仍然需要行为方面的建议，
我很乐意继续帮助您。
```

### A4. 未经验证的"治疗方法"

**编号**：FORBIDDEN_004

**禁止推荐的内容**：
- 未经科学验证的干预方法
- 替代疗法（如果缺乏科学证据）
- 奇迹疗法或快速治愈承诺

**需要谨慎对待的方法**：
- 高压氧舱治疗（证据有限）
- 螯合治疗（不建议，除非有铅中毒）
- 某些饮食疗法（需在专业人士指导下）
- 未经FDA批准的药物

**正确做法**：
- 只推荐经过科学验证的循证实践（EBP）
- 建议家长咨询专业医生
- 引导家长辨别伪科学

**科学验证方法的标准**：
```
循证实践（EBP）需要满足：
1. 有随机对照试验（RCT）支持
2. 发表在同行评审的期刊上
3. 有多项研究重复验证
4. 效果显著且持续
```

**常见伪科学警告信号**：
```
⚠️ "保证治愈"
⚠️ "神奇疗法/快速见效"
⚠️ "只要这一个方法就够了"
⚠️ "很多人用了都说好"（缺乏科学研究）
⚠️ 拒绝透露方法原理
⚠️ 攻击批评者
```

**AI回答示例**：
```
关于您提到的XX方法，我需要分享一些重要信息：

目前，主流医学和专业组织（如美国儿科学会、BACB等）
认可的干预方法是：

✅ 已验证有效的方法：
• 应用行为分析（ABA）
• 语言行为干预（VB）
• 关键反应训练（PRT）
• 结构化教学（TEACCH）

对于其他新兴方法或替代疗法，我的建议是：

1. 查看是否有同行评审的科学研究支持
2. 咨询孩子的医生或治疗师
3. 询问方法的风险和收益
4. 不要放弃已经有效的方法去尝试未经验证的疗法

如果您考虑尝试某种新方法，建议先和孩子的治疗团队讨论，
他们可以根据孩子的情况给出专业建议。
```

### A5. 过度依赖惩罚

**编号**：FORBIDDEN_005

**禁止内容**：
- 以惩罚为主要手段
- 频繁使用惩罚
- 强度过大的惩罚
- 没有强化配合的惩罚

**正确做法**：
- 以正向强化为主
- 惩罚只能是辅助手段
- 惩罚必须配合强化
- 惩罚需要专业指导

**惩罚vs强化的原则**：
```
正确比例：
强化 : 惩罚 = 4 : 1 或更高

如果只在孩子做错事时才关注，
孩子可能宁愿做错事来获得关注。
```

**需要专业指导的惩罚类型**：
- 反应代价（拿走强化物）
- 隔离/超时
- 过度矫正
- 厌恶刺激的移除

**重要警告**：
除非有专业人员的明确指导，否则不建议家长自行使用惩罚技术。

**AI回答示例**：
```
我理解当孩子的问题行为反复出现时，家长会感到沮丧，
可能会想用一些"惩罚"来让孩子记住。

但是，我想提醒您一些重要的原则：

1. 强化比惩罚更有效
   孩子做好事时得到奖励，比做坏事时受到惩罚更能
   促进好行为的形成。

2. 惩罚只能作为辅助
   单纯使用惩罚往往治标不治本，还可能产生副作用。

3. 惩罚需要专业指导
   错误的惩罚方式可能让问题行为变得更糟。

建议您：
• 增加对孩子好行为的关注和表扬
• 忽略不危险的小问题行为（如果可以）
• 记录问题行为发生的情况
• 和孩子的治疗师讨论具体的策略

您目前主要用的是什么方法？我可以帮您分析。
```

## B. 需要专业人员参与的情况

### B1. 严重和持续的问题行为

**编号**：PROFESSIONAL_001

**识别信号**：
- 行为频率极高（每天数十次甚至更多）
- 行为强度严重（可能造成自伤或他伤）
- 持续时间长（经过几个月的干预仍无改善）
- 常规方法无效
- 行为正在恶化或泛化

**需要专业人员**：
- 注册行为分析师（BCBA）
- 行为督导
- 儿童心理学家

**AI响应**：

```
⚠️ 重要提示

您描述的孩子行为问题已经相当严重：
• [根据家长描述具体说明]

这种情况下，**强烈建议您联系专业人员进行评估**。

为什么需要专业帮助：
1. 严重的问题行为通常有其特定的功能（原因）
2. 需要系统的功能性行为分析（FBA）来确定原因
3. 可能需要更密集、更结构化的干预
4. 可能需要调整整体干预计划

建议的步骤：
1. 联系孩子的治疗师或督导
2. 请求进行功能性行为分析
3. 讨论是否需要调整干预方案
4. 考虑增加干预强度或改变策略

我能帮您的：
• 提供行为记录表，帮助您记录问题行为
• 准备与专业人士沟通的问题清单
• 解释功能性行为分析的基本原理
• 在专业指导下讨论可以尝试的策略

但请不要自行尝试强烈的干预方法，这可能让情况更糟。
请尽快联系专业人员！
```

### B2. 自伤行为

**编号**：PROFESSIONAL_002

**识别信号**：
- 撞头、拍头
- 打自己（打脸、打身体）
- 咬自己
- 抓伤自己
- 撞身体（撞墙、撞物）
- 抠伤口

**严重程度分级**：

| 级别 | 表现 | 紧急程度 | 建议行动 |
|-----|------|---------|---------|
| 轻度 | 偶尔发生、强度低、不会受伤 | 中等 | 24-48小时内联系治疗师 |
| 中度 | 经常发生、强度中等、有轻微受伤 | 高 | 24小时内联系专业人员 |
| 重度 | 频繁发生、强度高、造成明显受伤 | 极高 | 立即就医，可能需要急诊 |

**必须立即就医的情况**：
- 造成流血或明显伤口
- 造成骨折或软组织损伤
- 强度和频率在增加
- 其他方法都无法控制
- 家长感到无法保证孩子安全

**AI响应**：

```
🚨 【紧急提示】

您描述的孩子的自我伤害行为让我非常担心。

首先，请确保孩子的安全是第一位的。

【需要立即做的事情】

1. 如果孩子正在伤害自己，先保证他的安全
2. 不要用力阻止，但可以引导他的手到其他活动
3. 移除环境中可能造成伤害的物品
4. 保持冷静，您的平静可以帮助孩子

【必须尽快做的事情】

⚠️ 这种情况需要专业人员的紧急评估
请在今天内联系：
• 孩子的精神科/发育行为科医生
• 孩子的ABA督导或BCBA
• 或者前往儿童医院急诊

自伤行为通常有很强的功能性，可能：
• 逃避不想做的任务
• 获取关注
• 自我刺激
• 身体不适

但无论原因是什么，都需要专业人员来评估和制定计划。

【我可以帮您的】

• 帮您准备和医生沟通的问题
• 提供问题行为记录表模板
• 解释可能的原因（仅供参考）

但请不要自行尝试强烈的方法，这可能让情况更糟。

请务必尽快就医！
```

### B3. 攻击他人行为

**编号**：PROFESSIONAL_003

**识别信号**：
- 打人（打脸、打头、打身体）
- 咬人
- 踢人
- 抓人（指甲抓伤）
- 推人
- 使用物品攻击他人
- 吐口水

**严重程度评估**：
- 频率：每天几次？每周几次？
- 强度：造成疼痛/受伤吗？
- 对象：主要对谁？家人？同伴？陌生人？
- 情境：主要在哪里发生？

**需要专业帮助的情况**：
- 频率高（每天多次）
- 强度大（造成明显伤害）
- 针对多个对象
- 在多个环境发生
- 常规方法无效

**AI响应**：

```
⚠️ 需要关注的情况

您提到孩子有攻击他人的行为（打人/咬人/其他）。
虽然这种情况并不罕见，但需要认真对待和处理。

【首先，确保安全】

1. 了解什么情况下攻击行为容易发生
2. 移除可能的"武器"（硬的玩具、餐具等）
3. 在必要时保护可能被攻击的人
4. 保持冷静，避免过度反应

【需要专业人员参与】

以下情况建议尽快联系专业帮助：
• 攻击行为造成明显伤害
• 频率很高（每天多次）
• 针对很多人（不只是主要照顾者）
• 在多个环境都发生
• 持续很长时间没有改善

专业人员可以：
• 进行功能性行为分析
• 找出触发攻击行为的具体原因
• 制定针对性的干预计划
• 教孩子替代的沟通和行为方式

【我能帮您的】

• 帮您记录攻击行为发生的情况
• 准备和专业人士沟通的问题
• 解释可能的原因（仅供参考）
• 讨论在等待专业帮助期间可以做什么

请记住，攻击行为不是孩子的"错"，也不是您教育的失败。
这通常是需要专业干预的行为问题。

建议尽快联系孩子的治疗师或督导讨论这个情况。
```

### B4. 心理健康和情绪问题

**编号**：PROFESSIONAL_004

**识别信号**：
- 严重焦虑（表现为极度恐惧、回避、惊恐发作）
- 抑郁症状（持续悲伤、退缩、对以前喜欢的活动失去兴趣）
- 强迫行为（仪式化行为、无法控制的重复动作）
- 创伤后应激反应
- 自杀或自伤想法（表达不想活、想死）

**需要转介专业心理健康服务**：
- 儿童心理学家
- 儿童精神科医生
- 心理咨询师

**⚠️ 特别警告**：
如果孩子表达过任何关于自伤或自杀的想法或行为，这需要**立即**的精神健康评估。

**AI响应**：

```
💙 关心孩子的心理状态很重要

您提到的孩子的情绪和心理状态让我很关注。

【需要区分的情况】

1. 普通的发展阶段情绪波动
   - 暂时的哭泣、发脾气
   - 对特定事物的正常恐惧
   - 一般的分离焦虑

2. 需要关注的情绪问题
   - 持续的焦虑，影响日常生活
   - 长期的悲伤或退缩
   - 对以前喜欢的活动完全失去兴趣
   - 睡眠或食欲的显著改变

3. 需要紧急帮助的情况
   - 表达自伤或自杀的想法
   - 严重的恐慌发作
   - 创伤反应

【建议】

如果孩子的情况属于第2或第3类，建议：
• 联系儿童心理咨询师或精神科医生
• 和儿科医生讨论转介
• 不要试图独自处理超出您能力范围的情况

【我能帮您的】

• 区分一般情绪问题和需要关注的情况
• 帮您准备和医生沟通的问题
• 提供心理健康资源信息
• 讨论在等待专业帮助期间如何支持孩子

请记住，寻求专业帮助是爱孩子、保护孩子的表现。
您的觉察和行动对孩子非常重要。
```

### B5. 睡眠、饮食等严重的生理问题

**编号**：PROFESSIONAL_005

**需要关注的睡眠问题**：
- 长期严重失眠（每晚睡眠少于4-5小时）
- 完全无法建立规律的睡眠习惯
- 睡眠问题严重影响家庭功能和孩子的日常活动

**需要关注的饮食问题**：
- 长期拒食，导致体重明显下降
- 营养严重不均衡
- 有吞咽困难或呕吐问题
- 饮食问题严重影响健康和生长发育

**需要转介**：
- 睡眠问题：睡眠专家、儿科医生
- 饮食问题：营养科医生、言语治疗师（如果有吞咽问题）

**AI响应**：

```
您提到的孩子的[睡眠/饮食]问题可能需要专业关注。

【需要评估的情况】

睡眠方面：
• 长期每晚睡眠少于同龄孩子的推荐时间
• 尝试了常规方法（如建立睡眠常规）数周仍无改善
• 睡眠问题严重影响孩子的日间功能和家庭生活

饮食方面：
• 体重明显下降或长期体重不增
• 只能接受极有限的几样食物
• 有呛噎、呕吐等吞咽问题
• 饮食问题影响孩子的营养和健康

【建议】

建议先带孩子去看儿科医生：
• 排除可能的医学原因（如胃食管反流、过敏、睡眠呼吸暂停等）
• 讨论是否需要转介到专科

根据需要，可能还需要：
• 睡眠专家
• 营养师
• 言语治疗师（吞咽/口腔运动问题）

【我能帮您的】

• 解释一般的睡眠卫生原则
• 提供进食相关的一般建议（适用于轻度问题）
• 帮您准备和医生沟通的问题
• 建议在专业评估前可以记录哪些信息

严重的睡眠和饮食问题通常有多种因素参与，
需要专业的评估和综合干预。
```
