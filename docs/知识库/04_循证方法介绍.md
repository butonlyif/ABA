## 什么是循证实践？

### 定义

循证实践是指经过科学研究验证，证明其有效的干预方法。

### 循证的标准

一个方法被认定为"循证实践"通常需要满足：
1. 有多项随机对照试验（RCT）支持
2. 发表在同行评审的期刊上
3. 有多项研究重复验证
4. 效果显著且可持续

### 主要的循证方法

| 方法 | 全称 | 主要特点 | 证据等级 |
|-----|------|---------|---------|
| DTT | 回合制教学 | 结构化、分解任务 | ★★★★★ |
| NET | 自然环境教学 | 自然情境中学习 | ★★★★☆ |
| VB | 语言行为 | 以语言为核心 | ★★★★☆ |
| PRT | 关键反应训练 | 游戏为基础 | ★★★★☆ |
| TEACCH | 结构化教学 | 视觉结构化 | ★★★★☆ |
| ESDM | 早期介入 | 综合方法 | ★★★★☆ |

## 一、DTT - 回合制教学

### 什么是DTT

DTT（Discrete Trial Training）是ABA中最经典、最系统的教学方法，也叫"回合制教学"或"离散试教"。

### 核心原理

DTT将技能分解成小的、独立的回合，每个回合包含：
- **指令**：给出明确的要求
- **辅助**：提供帮助使孩子做出正确反应
- **反应**：孩子做出行为
- **强化**：给予奖励

### DTT的结构

```
┌─────────────────────────────────────────┐
│               一个完整的DTT回合           │
├─────────────────────────────────────────┤
│                                         │
│  1. 前提/指令                           │
│     "指一指苹果"                         │
│                                         │
│  2. 辅助（如果需要）                     │
│     轻推孩子的手指向苹果                  │
│                                         │
│  3. 孩子的反应                          │
│     孩子指向苹果                          │
│                                         │
│  4. 强化                                │
│     "太棒了！给你饼干！"                 │
│     （立即给予强化物）                    │
│                                         │
│  5. 结果反馈                            │
│     记录反应正确/错误                     │
│                                         │
└─────────────────────────────────────────┘
```

### DTT的关键要素

#### 1. 明确的指令

```
好的指令：
✓ 简短清晰："指一指苹果"
✓ 一次一个要求
✓ 使用孩子的名字引起注意

不好的指令：
✗ "小明你看这个，这个是什么呀？
   是不是苹果？你来指一指好不好？"
```

#### 2. 及时辅助

辅助的层次：
```
层级1：完全身体辅助
  握着孩子的手完成整个动作

层级2：部分身体辅助
  在关键部分提供帮助

层级3：示范
  治疗师/家长示范一遍

层级4：手势提示
  用手势指向正确答案

层级5：言语提示
  "说：苹果"

层级6：独立
  孩子自己完成
```

#### 3. 即时强化

```
✓ 正确的反应 → 立即给强化物
✓ 强化物要孩子喜欢
✓ 配合口头表扬
✓ "你做到了！真棒！"
```

#### 4. 辅助撤退

```
原则：逐渐从多到少，从完全到不辅助

示例：教孩子说"苹果"

回合1：完全辅助
  辅助孩子说"苹果"，立即强化

回合2-5：部分辅助
  轻推孩子说出来，逐渐减少

回合6-10：示范
  家长说"苹果"，孩子仿说

回合11-15：言语提示
  "说..."，孩子接"苹果"

回合16+：独立
  "苹果"，孩子主动说
```

### DTT的优缺点

**优点**：
```
✓ 系统性强，目标清晰
✓ 便于数据记录
✓ 适合学习基础技能
✓ 可控性强
✓ 研究证据充分
```

**缺点**：
```
✗ 可能较刻板
✗ 泛化需要特别训练
✗ 缺乏自然情境
✗ 可能缺乏动机
✗ 人际互动较少
```

### DTT适用情况

**适合使用DTT的场景**：
- 学习新的基础技能
- 需要精确掌握的目标
- 技能分解后容易教授
- 初始学习阶段

**可能不适合的场景**：
- 需要自然泛化的技能
- 社交技能（需要人际互动）
- 日常生活的技能
- 孩子对DTT抗拒时

### DTT示例：教孩子命名

```
目标：孩子能说出图片中的10种水果名称

材料：水果图片卡

教学过程：

回合1：
指令："这是什么？"
辅助：治疗师示范说"苹果"
反应：孩子仿说"苹果"
强化："太棒了！这是苹果！"
      （立即给饼干）

回合2：
指令："这是什么？"
提示："苹..."
反应：孩子说"苹果"
强化："对了！苹果！"
      （给饼干）

回合3：
指令："这是什么？"
（无提示）
反应：孩子说"香蕉"
强化："你记住了！"
      （给饼干）

回合4：
指令："这是什么？"
反应：孩子沉默
强化：无（这是正确的"无反应"处理）
      （记录错误，继续下一个）
```

### 家长须知

```
✓ DTT是有效的教学方法，但需要专业培训
✓ DTT可以在家练习，但建议先学习正确方法
✓ DTT不是唯一的教学方法，需要结合其他方法
✓ 注意泛化，DTT中学的要在生活中练习
✓ 不要过度使用DTT，保持学习的趣味性
✓ 孩子状态不好时暂停
✓ 以愉快结束学习
```

### AI回答示例

## 二、NET - 自然环境教学

### 什么是NET

NET（Natural Environment Training）是与DTT相对的另一种教学方法，强调在自然情境中学习。

### 核心原理

NET利用孩子自然的动机，在日常活动和游戏中有机地融入教学，而不是在固定的桌面进行。

### NET vs DTT

```
┌─────────────────┬─────────────────┐
│      DTT         │      NET         │
├─────────────────┼─────────────────┤
│ 固定桌面教学     │ 自然情境中学习    │
│ 分解的任务      │ 自然发生的机会    │
│ 人工的强化物     │ 自然强化物       │
│ 结构化          │ 灵活性高         │
│ 成人主导        │ 孩子主导         │
└─────────────────┴─────────────────┘
```

### NET的核心策略

#### 1. 利用自然动机

```
不是："现在我们学颜色，坐下。"
而是："（看到红色的球）球！什么颜色？"
```

#### 2. 跟随孩子的兴趣

```
孩子正在玩车：
不是："放下车，我们学认知"
而是：跟随孩子，在玩车中学习
```

#### 3. 在自然情境中嵌入教学

```
吃饭时：
- 教命名（这是什么？）
- 教提要求（你想要什么？）
- 教社交（谢谢、你好）

洗澡时：
- 教身体部位
- 教动词（洗、擦）
- 教指令服从
```

### NET的具体技术

#### 1. 随机教学

在自然游戏中随机嵌入教学目标：

```
场景：孩子正在玩积木

治疗师/家长等待机会...

孩子："塔"（自言自语）

治疗师："塔？搭得很高！"
        "这是什么？"（指向红色积木）
        ↓
        教学回合

孩子继续玩...
        ↓
        又一个机会
        ↓
        教学回合
```

#### 2. 诱发性教学

创造机会诱发教学目标：

```
目标：教孩子说"还要"

步骤：
1. 给孩子几块饼干
2. 吃掉几块后，藏着剩下的
3. 孩子想要更多
4. 等待孩子表达
5. 如果孩子不说话，提示"还要"
6. 孩子说"还要"后，立即给饼干

这就是利用自然动机（想吃饼干）教学
```

#### 3. 悬置（Mand-titude）

等待孩子表达需求，然后教学：

```
孩子看着饼干罐

不是立即给他
↓ 等待
↓ 观察孩子是否表达
↓ 如果孩子指/说
→ 立即满足
→ "你说了'饼干'！给你！"

如果孩子不表达
→ 提示"说'饼干'"
→ 孩子仿说
→ 立即给
```

### NET的优缺点

**优点**：
```
✓ 自然情境，泛化容易
✓ 孩子更有动机
✓ 更接近真实生活
✓ 培养社交互动
✓ 孩子主导，更有趣
```

**缺点**：
```
✗ 教学机会可能不规律
✗ 难以精确控制
✗ 数据记录较难
✗ 需要更高的技巧
✗ 不适合初始学习
```

### NET适用情况

**适合使用NET的场景**：
- 泛化学过的技能
- 社交技能
- 自然情境中的技能
- 维持已学技能
- 增加学习动机

**可能不适合的场景**：
- 初始学习新技能
- 需要精确掌握的目标
- 高度结构化的技能

### NET示例：教社交技能

```
目标：孩子能在游戏中主动发起互动

场景：治疗师和3个孩子一起玩追逐游戏

治疗师：追着孩子跑
        （等待孩子发起互动的机会）

孩子A：停下来转身看治疗师
       （这是社交发起！）

治疗师：立即回应
        "你好！"
        "我们继续玩！"
        （强化孩子的社交发起）

孩子B：没有反应
       治疗师停止追逐
       （制造机会）

孩子B：伸手想继续
       （被动互动）

治疗师：回应并强化
        "你要继续！好！"

孩子C：主动说"追我！"
       （主动社交发起！）

治疗师：大大的强化
        "你说'追我'！太好玩了！"
```

### NET家长须知

```
✓ NET可以融入日常生活
✓ 在游戏、吃饭、出行中都可以练习
✓ 跟随孩子的兴趣
✓ 等待自然的机会
✓ 不要太刻意，保持有趣
✓ 和DTT结合使用效果最好
```

```

【举个例子】
```

## 三、VB - 语言行为

### 什么是VB

VB（Verbal Behavior，语言行为）是以斯金纳的语言理论为基础，强调语言的功能性使用。

### 核心原理

VB认为语言是一种行为，应该根据其功能来分类：

```
┌─────────────────────────────────────────┐
│          VB语言分类                       │
├─────────────────────────────────────────┤
│                                         │
│  提要求 (Mand)                          │
│  "我要苹果"                              │
│  孩子表达需求                             │
│                                         │
│  命名 (Echoic)                          │
│  "苹果"                                  │
│  模仿/复述语言                           │
│                                         │
│  命名 (Mand)                            │
│  "这是什么？"→"苹果"                    │
│  用语言标签事物                           │
│                                         │
│  对话 (Intraverbal)                     │
│  "你叫什么名字？"→"小明"                │
│  回答问题、会话                          │
│                                         │
└─────────────────────────────────────────┘
```

### VB的四种语言操作

#### 1. 提要求 (Mand)

**定义**：孩子用语言表达需求

**为什么重要**：
- 功能性最强
- 减少问题行为
- 满足需求

**如何教**：
```
步骤1：创造动机
  把孩子想要的饼干放在看得见拿不到的地方

步骤2：等待表达
  等待孩子表示想要

步骤3：提示
  如果孩子不说，提示"说'我要饼干'"

步骤4：强化
  孩子说"我要饼干"，立即给饼干

步骤5：泛化
  用不同的强化物练习
```

#### 2. 仿说 (Echoic)

**定义**：模仿/复述听到的语言

**为什么重要**：
- 学习语言的基础
- 扩大语言库
- 为自发语言打基础

**如何教**：
```
步骤1：获得注意力
  "看我说"

步骤2：说一个字/词
  "苹果"

步骤3：提示仿说
  "你说：苹果"

步骤4：强化
  "你说对了！"

步骤5：扩展
  逐渐增加词的数量
```

#### 3. 命名 (Receptive/ Tact)

**定义**：用语言标签事物

**命名 vs 提要求**：
```
命名：看到苹果 → "苹果"（物体→语言）
提要求：想吃苹果 → "苹果"（语言→物体）
```

**如何教命名**：
```
步骤1：呈现刺激物
  展示苹果图片

步骤2：提问
  "这是什么？"

步骤3：提示（如果需要）
  示范"苹果"

步骤4：强化
  "对了！苹果！"

步骤5：泛化
  用不同的苹果图片、实物练习
```

#### 4. 对话性语言 (Intraverbal)

**定义**：回答问题和参与对话

**为什么难**：
- 需要理解问句
- 需要提取记忆
- 需要自发组织语言

**如何教**：
```
简单问答（开始）：
问："你叫什么名字？" → "小明"
问："你想要什么？" → "苹果"
问："你喜欢什么？" → "车车"

进阶问答：
问："小狗怎么说？" → "汪汪"
问："妈妈给你买什么？" → "玩具"

对话：
A："你今天去哪里了？"
B："去了公园"
A："公园有什么好玩的？"
B："荡秋千"
```

### VB教学策略

#### 错误纠正程序 (Error Correction)

```
孩子答错了 → 不是简单说"不对"

而是：

步骤1：重新教学
  "不是红色的，是蓝色的"

步骤2：再次呈现
  "这是什么颜色？"

步骤3：再次辅助
  强调"蓝..."

步骤4：再次尝试
  "蓝色！对了！"
```

#### 渐进式prompting

```
不是一次性提示：
  "指红色的，不是蓝色的"（太多信息）

而是逐步提示：
  "指红色的" → 如果错
  "不是蓝色的" → 如果还错
  "最上面那个" → 逐步缩小范围
```

### VB的优缺点

**优点**：
```
✓ 强调语言的功能性
✓ 减少问题行为
✓ 增强沟通能力
✓ 有系统的评估工具（VB-MAPP）
✓ 适合语言障碍儿童
```

**缺点**：
```
✗ 技术要求高
✗ 需要专业培训
✗ 可能过于强调语言
✗ 部分技能需要其他方法配合
```

### VB适用情况

**VB适合**：
- 语言发展迟缓
- 缺乏功能性语言
- 自闭症谱系障碍
- 想要提升沟通能力

**VB-MAPP评估**：
VB-MAPP是评估语言能力的工具，可以：
- 评估当前语言水平
- 识别语言障碍
- 制定教学目标
- 追踪进步

### VB家长须知

```
✓ 语言的核心是功能性沟通
✓ 提要求是最重要的语言技能
✓ 在日常生活中创造语言机会
✓ 强化所有的语言尝试
✓ 等待孩子主动表达
✓ 泛化到不同情境和人
✓ 可以和VB-MAPP评估结合使用
```

```
问：什么是VB语言行为？

答：

【VB的核心理念】

   → 用语言标签事物

4. 对话
   问答"你叫什么？"→"小明"
   → 参与会话

【为什么提要求最重要】

因为：
✓ 提要求能直接满足孩子的需求
✓ 减少问题行为（不用哭闹来得到）
✓ 是功能性最强的语言
✓ 是沟通的基础

【日常生活中怎么做】

✓ 创造提要求的机会
  把孩子想要的东西放高一点

✓ 等待孩子说
  不要孩子一指就给

✓ 立即强化
  说了就给

✓ 在日常中练习命名和对话
  看到什么说什么

VB很适合提升自闭症孩子的语言能力！
```

## 四、PRT - 关键反应训练

### 什么是PRT

PRT（Pivotal Response Training，关键反应训练）是以游戏为基础、自然情境中的干预方法。

### 核心原理

PRT针对"关键性"技能，这些技能提高了，其他技能也会跟着进步。

### 什么是"关键"技能

PRT聚焦于以下关键领域：

```
┌─────────────────────────────────────────┐
│           PRT的关键技能                   │
├─────────────────────────────────────────┤
│                                         │
│  1. 动机                                │
│     → 学习动机强了，其他都更容易         │
│                                         │
│  2. 对多种线索的反应                      │
│     → 能对不同的要求做出反应             │
│     → 泛化能力增强                       │
│                                         │
│  3. 自我管理                             │
│     → 学会自我监控和调节                 │
│                                         │
│  4. 社交发起                            │
│     → 主动和他人互动                     │
│                                         │
└─────────────────────────────────────────┘
```

### PRT的核心策略

#### 1. 激发动机

```
策略1：任务选择
  让孩子选择想学什么
  "你想先学颜色还是形状？"

策略2：交替新旧任务
  新任务（难）+ 旧任务（简单/已会）
  保持兴趣和成功率

策略3：自然强化
  学说话"苹果" → 得到苹果（自然结果）
  而不是学说话"苹果" → 得到贴纸（人为强化）

策略4：奖励努力
  "你很努力！"（而不是只说"对了"）
```

#### 2. 维持和扩展

```
不是学一个就换一个：
✓ 学会后继续练习
✓ 变化刺激物（颜色、大小不同的苹果）
✓ 变化情境（超市、家里、公园）

→ 增强泛化能力
```

#### 3. 教社交发起

```
不是孩子被动回应：
✓ 创造社交机会
✓ 等待孩子主动发起
✓ 大大强化社交发起

孩子主动说"看我的车" → 立即回应和强化
```

#### 4. 自我管理（进阶）

```
让孩子学会自我监控：

步骤1：教孩子识别目标行为
  "你在和小朋友玩吗？"

步骤2：教自我记录
  完成任务后，自己打勾

步骤3：教自我强化
  完成目标后，自己奖励自己
```

### PRT的4个关键领域详解

#### 1. 动机

**提高动机的技巧**：
```
✓ 让孩子选择
✓ 任务穿插（难+易）
✓ 使用自然强化
✓ 奖励努力，不只奖励结果
✓ 保持任务有趣
✓ 建立信任和关系
```

**为什么动机重要**：
```
动机低 → 孩子不配合 → 学习效果差
动机高 → 孩子主动 → 学习效果好

提高动机后，其他技能学习都更容易！
```

#### 2. 对多种线索的反应

**什么是"多种线索"**：
```
单一线索：
指令："指红色的球"（只有一个线索）

多种线索：
指令："指红色的球"（颜色+物品）
指令："把红色的球给妈妈"（颜色+物品+人）

能对多种线索反应 → 泛化能力强
```

**如何训练**：
```
✓ 逐渐增加线索数量
✓ 从单一线索开始，逐渐叠加
✓ 在不同情境中练习
✓ 用不同的表达方式
```

#### 3. 自我管理

**为什么重要**：
```
自我管理强的孩子：
✓ 能自我调节情绪
✓ 能监控自己的行为
✓ 更独立
✓ 为成年独立打基础
```

**如何教（根据孩子能力）**：
```
基础级：
  教孩子识别自己的情绪
  "你开心吗？"

中级：
  教自我记录
  "做到了就画个圈"

进阶：
  教自我监控和调节
  "你刚才发脾气，现在呢？"
  "可以怎么让自己平静下来？"
```

#### 4. 社交发起

**什么是社交发起**：
```
被动社交：
  别人问，孩子答
  （被动的反应性互动）

主动社交：
  孩子主动和他人说话/互动
  （主动的自发性互动）
```

**如何提高社交发起**：
```
✓ 创造社交机会
✓ 等待孩子主动
✓ 大大强化主动社交
✓ 示范社交发起
✓ 角色扮演练习
```

### PRT的优缺点

**优点**：
```
✓ 游戏为基础，孩子喜欢
✓ 自然情境，泛化容易
✓ 聚焦关键技能，效果大
✓ 适合家庭使用
✓ 研究证据充分
```

**缺点**：
```
✗ 可能不够结构化
✗ 不适合初始学习很新的技能
✗ 需要较高的技巧
✗ 数据记录较难
```

### PRT适用情况

**PRT适合**：
- 社交技能
- 语言沟通
- 动机问题
- 泛化困难
- 家庭干预

**PRT不适合**：
- 需要精确掌握的基础技能
- 高度结构化的学习目标
- 初始学习全新技能

### PRT家长须知

```
✓ PRT很适合在家使用
✓ 在游戏中练习
✓ 提高孩子的学习动机
✓ 让孩子有选择
✓ 等待孩子主动
✓ 大大强化主动发起
✓ 变化不同的情境和人
✓ 保持有趣和轻松
```

### AI回答示例


## 五、TEACCH - 结构化教学

### 什么是TEACCH

TEACCH（Treatment and Education of Autistic and Communication related Handicapped children）是专门为自闭症设计的结构化教学方法。

### 核心原理

TEACCH利用自闭症孩子的特点（视觉处理优势、系统化偏好），通过结构化帮助他们理解世界。

### 结构化的四个维度

```
┌─────────────────────────────────────────┐
│         TEACCH的四个结构维度             │
├─────────────────────────────────────────┤
│                                         │
│  1. 物理结构                            │
│     环境的组织和安排                     │
│                                         │
│  2. 作息/时间结构                        │
│     可视化的日程安排                     │
│                                         │
│  3. 工作/活动结构                        │
│     任务的组织和呈现                     │
│                                         │
│  4. 个别化结构                           │
│     根据孩子特点定制                     │
│                                         │
└─────────────────────────────────────────┘
```

### 1. 物理结构

**什么是物理结构**：
```
环境的组织和分区

示例：教室分区

┌─────────┬─────────┬─────────┐
│ 个人区域 │  团体区  │  过渡区  │
│         │         │         │
│ 独立工作  │  集体活动  │ 等待/休息│
│ 桌子     │  地毯     │  沙发    │
│ 减少干扰  │  社交互动  │  感官调节│
└─────────┴─────────┴─────────┘
```

**为什么重要**：
```
✓ 帮助孩子理解不同区域的功能
✓ 减少混乱和焦虑
✓ 提示孩子该做什么
✓ 减少问题行为
```

**如何在家实施**：
```
✓ 明确划分区域（玩耍区、学习区、休息区）
✓ 用视觉标记区分
✓ 减少视觉干扰
✓ 创设安静的学习角
```

### 2. 作息/时间结构

**什么是作息结构**：
```
可视化的日程安排

示例：每日日程表

┌────┬────┬────┬────┬────┬────┬────┐
│起床│早餐│学习│游戏│午餐│午休│户外│
└────┴────┴────┴────┴────┴────┴────┘
```

**时间表示方式**：

| 方式 | 适用年龄 | 示例 |
|-----|---------|-----|
| 照片/图片 | 幼儿 | 📷→👕→🍎 |
| 图文卡片 | 稍大 | 📚→🎮→🍽 |
| 文字时间表 | 大龄 | "学习" |
| 计时器 | 所有人 | 沙漏、数字计时器 |

**如何使用**：
```
✓ 每项活动有图片/文字
✓ 完成的划掉/拿走
✓ 预告下一个活动
✓ 帮助孩子理解"接下来是什么"
```

### 3. 工作/活动结构

**什么是工作结构**：
```
任务的可视化组织

示例：任务盒

┌─────────────────────────┐
│      做什么（左边）       │
│    📷 👕 🍎 拼图         │
│    （任务的图片）         │
├─────────────────────────┤
│                         │
│        怎么做（中间）     │
│    ┌─────┬─────┬─────┐  │
│    │ 1   │ 2   │ 3   │  │
│    │ 📷  │ 👕  │ 🍎  │  │
│    │ 完成 │ 完成 │ 完成 │  │
│    └─────┴─────┴─────┘  │
│    （步骤和顺序）          │
│                         │
├─────────────────────────┤
│      到哪里（右边）       │
│        🎁              │
│    （完成后的奖励）        │
│                         │
└─────────────────────────┘
```

**任务结构的要点**：
```
✓ 清楚标明"做什么"
✓ 清楚标明"怎么做"（步骤顺序）
✓ 清楚标明"做完去哪"（结果）
✓ 适量任务，不堆积
✓ 难度适中
```

### 4. 个别化结构

**什么是个别化**：
```
根据每个孩子的能力、特点来定制

考虑因素：
✓ 能力水平
✓ 感官偏好
✓ 学习风格
✓ 兴趣
✓ 优势
✓ 挑战
```

### TEACCH的核心策略

#### 视觉结构

```
自闭症孩子通常是视觉学习者：

✓ 视觉比听觉更有效
✓ 图像比文字更清晰
✓ 展示比说明更直观

✓ 用图片展示任务
```

#### 一致性和预测性

```
✓ 固定的日程安排

✓ 固定的工作流程
✓ 减少意外和变化

```

#### 系统化

```
✓ 任务分解成小步骤
✓ 清晰的任务顺序
✓ 明确的开始和结束
✓ 有组织的环境
```

### TEACCH的优缺点

**优点**：
```
✓ 非常适合自闭症特点
✓ 减少焦虑和混乱
✓ 提高独立性
✓ 可视化，易理解
✓ 可在家里使用
✓ 证据充分

```

```
✗ 不够灵活
✗ 需要准备材料
✗ 过渡到自然情境需要额外努力
```

### TEACCH适用情况

**TEACCH适合**：
- 自闭症谱系障碍
- 需要结构化的孩子
- 容易焦虑/混乱的孩子
- 需要提高独立性的孩子

**在家里特别有用**：
- 建立日常规律
- 组织学习角
- 任务可视化
- 减少混乱

### TEACCH家长须知

```

✓ 保持灵活性
```

## 六、其他循证方法

### ESDM - 早期丹佛模式

**什么是ESDM**：
ESDM（Early Start Denver Model）是针对1-3岁自闭症幼儿的综合早期干预方法，结合了DTT和PRT的优点。

**核心特点**：
```
✓ 以关系为基础
✓ 游戏为导向
✓ 早期密集干预
✓ 综合多领域技能
✓ 家长参与
```

**适用年龄**：
- 12-48个月
- 越早越好

**证据**：
- 有RCT研究支持
- 能改善认知和语言

---

### ABI - 咬合法

**什么是ABI**：
ABI（Autism Biomodule Intervention）是一种使用口腔运动工具来改善口腔运动功能和言语的干预方法。

**核心特点**：
```
✓ 针对性口腔运动
✓ 使用专业的咬合工具
✓ 改善口腔感觉
✓ 促进发音
```

**适用情况**：
- 口腔运动问题
- 构音障碍
- 口腔敏感

---

### SSB - 感统训练

**什么是SSB**：
SSB（Sensory Integration）是感觉统合训练，通过提供适当的感官输入来帮助孩子处理感官信息。

**核心感觉系统**：
```
✓ 触觉（触摸、质地）
✓ 前庭觉（平衡、空间感）
✓ 本体觉（身体位置、力量）
✓ 听觉（声音）
✓ 视觉（看）
✓ 嗅觉/味觉
```

**感统失调的表现**：
```
感觉过敏：
- 讨厌某种材质
- 对声音过度反应

感觉迟钝：
- 对疼痛没反应
- 需要强烈的感官刺激

感觉寻求：
- 喜欢转圈、跳
- 寻求感官输入
```

**感统训练策略**：
```
✓ 荡秋千（前庭觉）
✓ 推拉重物（本体觉）
✓ 触摸不同材质（触觉）
✓ 跳跳床（本体的）
✓ 挤压、拥抱（触觉）
```

---

### 社会故事

**什么是社会故事**：
社会故事（Social Stories）是用于教自闭症孩子理解和应对社交情境的工具。

**结构**：
```
社会故事格式：

标题：去餐厅吃饭

描述：
我们去餐厅吃饭。餐厅里有桌子和椅子。

观点：
我可能会有点紧张，但是没关系。

步骤：
1. 我们走进餐厅
2. 找到座位坐下
3. 点餐
4. 等待食物
5. 吃饭
6. 和服务员说谢谢
7. 离开

结尾：
我们去餐厅吃完饭，很开心。
```

**如何写社会故事**：
```
✓ 用孩子的名字
✓ 用第一人称
✓ 正面描述
✓ 简短具体
✓ 配图片更好
✓ 强调适当的行為
✓ 练习和复习
```

---

### 同伴介入

**什么是同伴介入**：
同伴介入（Peer-Mediated Intervention）是训练普通孩子和自闭症孩子互动的方法。

**怎么做**：
```
1. 培训普通孩子
   - 了解自闭症特点
   - 学习简单的互动技巧
   - 练习如何邀请

2. 创造互动机会
   - 游戏时间
   - 小组活动
   - 社交练习

3. 支持和反馈
   - 成人监督
   - 即时反馈
   - 强化好的互动
```

**效果**：
```
✓ 提高社交技能
✓ 增加社交发起
✓ 改善同伴关系
✓ 促进融合
```

## 方法选择指南

### 选择方法考虑因素

```
选择干预方法时考虑：

1. 孩子的年龄
   - 幼儿 → ESDM、DTT
   - 任何年龄 → PRT、NET

2. 孩子的能力
   - 初始学习 → DTT
   - 泛化困难 → NET、PRT

3. 目标技能
   - 语言 → VB、DTT
   - 社交 → PRT、TEACCH
   - 自理 → DTT、NET

4. 家庭情况
   - 家长能力
   - 时间资源
   - 家庭偏好

5. 专业支持
   - 有无BCBA督导
   - 治疗师经验
```

### 方法组合建议

```
单一方法可能不够，建议组合：

组合1：
DTT（学新技能）+ NET（泛化）

组合2：
VB（语言）+ PRT（社交）+ TEACCH（结构）

组合3：
ESDM（综合早期）+ 感统（感觉支持）

组合4：
DTT（桌面）+ 社会故事（社交理解）
+ 同伴介入（实践社交）
```

### 年龄建议

```
0-3岁：
✓ ESDM（早期综合）
✓ DTT（基础技能）
✓ PRT（游戏和动机）

3-6岁：
✓ DTT + NET组合
✓ VB（语言）
✓ TEACCH（结构）
✓ PRT（社交）

6岁以上：
✓ NET为主
✓ PRT（社交）
✓ TEACCH（学业支持）
✓ 社会故事
```
