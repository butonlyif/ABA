## 使用说明

本文档面向家长日常高频场景，采用统一结构，便于AI检索和生成可执行建议。

每个场景包含：

- **适用情况**：什么问题可以参考
- **先排除**：需要先确认的医学、安全或环境因素
- **可能功能**：行为可能在满足什么需求
- **家长当下做法**：事件发生时怎么处理
- **提前预防**：下次如何降低发生概率
- **替代技能**：教孩子用什么更合适的方式表达
- **强化策略**：如何强化目标行为
- **记录指标**：家长可以记录什么
- **转介信号**：什么时候必须找专业人员

重要原则：所有建议均为一般性家庭支持，不能替代医生、BCBA、语言治疗师或作业治疗师的个别化评估。

## SCENE_001 发脾气、哭闹、倒地

**适用情况**：孩子因为被拒绝、等待、切换活动、要求未满足而哭闹、尖叫、倒地。

**先排除**：
- 是否饥饿、困倦、疼痛、便秘、发烧或身体不适
- 是否环境太吵、太亮、人太多
- 是否任务过难或等待时间过长

**可能功能**：
- 获得想要的物品或活动
- 逃避不喜欢的任务
- 获得家长关注
- 感官过载后的自我调节

**家长当下做法**：
1. 保持孩子和周围人的安全，减少围观和语言刺激。
2. 用短句说明边界，例如：“现在不能买糖。你可以选水或饼干。”
3. 如果行为是为了获得物品，不在哭闹高峰时让步。
4. 等孩子稍微平静后，提示一个简单替代表达：“说‘帮我’”“指一指”。
5. 一旦孩子用替代方式表达，立即回应并表扬。

**提前预防**：
- 出门前说明流程和可选项
- 使用视觉日程、倒计时、先后板
- 对容易触发哭闹的场景提前练习

**替代技能**：
- 说“我还想要”
- 说“休息一下”
- 使用“等待卡”“帮助卡”“不要卡”

**强化策略**：
- 强化平静等待、用语言/手势表达、接受替代选项
- 不强化尖叫、倒地后立即获得原本想要的东西

**记录指标**：
- 每天发生次数
- 持续时间
- 触发事件
- 最后是否用替代方式表达

**转介信号**：
- 行为持续升级、每天多次且影响生活
- 出现自伤、攻击或危险逃跑
- 家长无法保证安全

## SCENE_002 打人、推人、咬人

**适用情况**：孩子打家长、推同伴、咬人、抓人或扔东西伤人。

**先排除**：
- 是否有疼痛、牙痛、耳部不适或睡眠严重不足
- 是否被同伴靠太近、抢玩具或环境过载

**可能功能**：
- 逃离不喜欢的人或任务
- 获得玩具、空间或关注
- 表达不会说的拒绝、愤怒、害怕

**家长当下做法**：
1. 先挡住攻击，拉开距离，语气平静。
2. 用短句标记：“不能打人。手放下。”
3. 不长篇说教，不在攻击后给大量关注。
4. 引导替代行为：“说‘不要’”“说‘走开’”“把卡片给我”。
5. 一旦孩子用替代方式表达，立即回应。

**提前预防**：
- 教“不要”“停”“轮到我”“帮我”
- 对容易冲突的玩具准备轮流规则
- 用计时器练习轮流和等待

**替代技能**：
- 推开前先说“不要”
- 想要物品时说“给我玩”
- 不舒服时说“离我远一点”

**强化策略**：
- 强化手放好、用语言拒绝、轮流等待
- 对轻微攻击后的长篇安抚要谨慎，避免强化攻击

**记录指标**：
- 攻击对象
- 前一刻发生了什么
- 攻击后孩子获得了什么或逃掉了什么

**转介信号**：
- 攻击造成受伤
- 攻击频率高或无法预测
- 咬人、撞头、拿危险物品攻击

## SCENE_003 自伤：撞头、打自己、咬自己

**适用情况**：孩子撞头、打脸、咬手、抓破皮肤。

**先排除**：
- 立即检查是否有伤口、疼痛、牙痛、耳痛、头痛、癫痫或其他医学因素
- 自伤行为需要优先考虑医学评估和专业行为评估

**可能功能**：
- 逃避痛苦或任务
- 表达极端情绪
- 获得强烈感官刺激
- 获得关注或帮助

**家长当下做法**：
1. 立即保护身体安全，移除硬物和尖锐物。
2. 降低语言刺激，使用简短稳定的句子。
3. 不责骂、不羞辱、不强行讲道理。
4. 平静后提供替代表达：“疼”“不要”“休息”“帮我”。
5. 尽快联系专业人员做功能评估。

**提前预防**：
- 记录自伤前的具体触发
- 降低任务难度
- 提供可接受的感官替代物
- 建立休息请求方式

**替代技能**：
- 用休息卡请求暂停
- 用疼痛卡表达身体不适
- 用压力球、抱枕等安全替代物释放

**强化策略**：
- 强化提前求助、暂停、离开刺激环境
- 不让自伤成为唯一能获得帮助的方式

**记录指标**：
- 自伤部位和强度
- 持续时间
- 是否受伤
- 前事和后果

**转介信号**：
- 任何造成伤害的自伤都建议尽快专业介入
- 若有出血、意识异常、严重撞击，立即就医

## SCENE_004 抢玩具、不愿轮流

**适用情况**：孩子抢同伴玩具、轮到别人时哭闹、不能等待。

**先排除**：
- 是否规则过难或等待时间过长
- 是否孩子还不会表达“我想玩”

**可能功能**：
- 获得物品
- 避免等待
- 不理解轮流规则

**家长当下做法**：
1. 先挡住抢夺，简短说明：“现在是哥哥玩。”
2. 立即提示替代表达：“说‘我想玩’。”
3. 给明确等待时间：“等计时器响，轮到你。”
4. 等待成功哪怕2秒也立即强化。

**提前预防**：
- 从非常短的等待开始
- 使用计时器、轮流卡
- 选择孩子不那么执着的玩具练习

**替代技能**：
- “轮到我了吗？”
- “我想玩”
- “换一下”

**强化策略**：
- 强化等待、询问、交换、把玩具还给别人
- 初期等待时间要短，成功率优先

**记录指标**：
- 最长可等待时间
- 是否需要提示
- 是否能归还玩具

**转介信号**：
- 因抢夺频繁攻击他人
- 轮流训练长期没有进展

## SCENE_005 切换活动困难

**适用情况**：从玩具切到吃饭、从动画片切到洗澡、从家里出门时崩溃。

**先排除**：
- 是否提前预告不足
- 是否下一个活动太难或太不喜欢

**可能功能**：
- 继续喜欢的活动
- 逃避下一个活动
- 对不确定性感到焦虑

**家长当下做法**：
1. 使用“先-后”句式：“先收玩具，后看书。”
2. 给一个明确选择：“你拿小车去洗澡，还是拿鸭子？”
3. 避免突然抢走物品。
4. 切换成功立即表扬并给小强化。

**提前预防**：
- 用视觉日程
- 提前5分钟、2分钟、30秒预告
- 给孩子参与权，例如自己关电视

**替代技能**：
- “再玩一分钟”
- “我需要帮忙”
- “我想带一个玩具”

**强化策略**：
- 强化听到预告后开始准备
- 强化自己关掉/放回物品

**记录指标**：
- 哪类切换最难
- 提前预告后是否改善
- 切换所需时间

**转介信号**：
- 每次切换都严重崩溃
- 出现攻击、自伤或逃跑

## SCENE_006 不听指令、叫不动

**适用情况**：家长叫孩子名字、让他坐下、收玩具、洗手，孩子没反应。

**先排除**：
- 听力问题
- 指令太长、太复杂
- 孩子正在高度沉浸活动

**可能功能**：
- 没理解
- 没动机
- 逃避任务
- 注意力没有转向家长

**家长当下做法**：
1. 先靠近孩子，叫名字，等待看向你。
2. 一次只给一个短指令：“拿鞋。”
3. 必要时示范或手势提示。
4. 做到后立即具体表扬。

**提前预防**：
- 练习名字反应
- 把复杂任务拆小
- 给指令前减少干扰

**替代技能**：
- 学会回应“来了”
- 学会请求“等一下”

**强化策略**：
- 强化第一次回应
- 强化看向家长后执行

**记录指标**：
- 第一次指令成功率
- 需要重复几次
- 哪类指令最困难

**转介信号**：
- 对声音持续无反应
- 理解能力明显落后，需评估

## SCENE_007 挑食、拒绝新食物

**适用情况**：只吃少数食物、拒绝新食物、闻到或看到就哭。

**先排除**：
- 吞咽困难、过敏、胃肠问题、口腔肌肉问题
- 体重下降、营养不足需就医

**可能功能**：
- 逃避不喜欢的味道/口感
- 感官敏感
- 获得熟悉食物

**家长当下做法**：
1. 不强塞、不威胁。
2. 新食物目标从“看一眼、碰一下、闻一下”开始。
3. 熟悉食物和新食物一起出现。
4. 对任何接近行为立即表扬。

**提前预防**：
- 固定餐桌流程
- 少量多次暴露
- 从相似口感开始

**替代技能**：
- “不要”
- “小口”
- “喝水”

**强化策略**：
- 强化看、碰、闻、舔、小口尝
- 不把大量甜食作为唯一替代

**记录指标**：
- 可接受食物清单
- 新食物接近层级
- 每次尝试强度

**转介信号**：
- 食物种类极少、体重异常、呕吐、吞咽困难

## SCENE_008 睡前拖延、入睡困难

**适用情况**：睡前兴奋、反复下床、要求玩手机、很晚才睡。

**先排除**：
- 睡眠呼吸问题、疼痛、药物影响
- 白天小睡过长、睡前屏幕刺激

**可能功能**：
- 延迟睡觉
- 获得家长陪伴或手机
- 逃避独处

**家长当下做法**：
1. 固定睡前流程，少说话。
2. 反复下床时平静带回床，不展开互动。
3. 使用视觉睡前流程卡。
4. 早上强化“昨晚按流程睡觉”。

**提前预防**：
- 睡前30-60分钟减少屏幕
- 固定洗漱、故事、关灯顺序
- 白天安排足够运动

**替代技能**：
- “抱一下”
- “讲一个故事”
- “我害怕”

**强化策略**：
- 强化按流程完成每一步
- 强化留在床上

**记录指标**：
- 上床时间
- 入睡时间
- 夜醒次数

**转介信号**：
- 长期严重失眠、打鼾、夜间异常行为

## SCENE_009 如厕训练困难

**适用情况**：拒绝坐马桶、不会表达要上厕所、频繁尿裤子。

**先排除**：
- 便秘、尿路感染、腹痛
- 是否具备基本坐一会、简单配合能力

**可能功能**：
- 逃避马桶
- 不理解身体信号
- 对马桶声音或感觉敏感

**家长当下做法**：
1. 不羞辱尿裤子。
2. 固定时间短暂坐马桶。
3. 成功排尿/排便立即高强度强化。
4. 用图片或词教“尿尿”“便便”。

**提前预防**：
- 记录排尿/排便时间规律
- 准备脚凳、儿童马桶圈
- 从坐5秒开始逐渐增加

**替代技能**：
- 指厕所图片
- 说“尿尿”
- 拉家长去厕所

**强化策略**：
- 强化走进厕所、坐马桶、表达需求、成功排泄

**记录指标**：
- 干裤时间
- 成功次数
- 坐马桶配合度

**转介信号**：
- 疑似便秘、疼痛、频繁尿路问题

## SCENE_010 公共场所崩溃

**适用情况**：超市、餐厅、医院、地铁等场所哭闹、逃跑、尖叫。

**先排除**：
- 噪音、灯光、人群、气味导致的感官过载
- 等待时间是否过长

**可能功能**：
- 离开环境
- 获得想要物品
- 逃避等待
- 感官调节

**家长当下做法**：
1. 优先安全，靠边或到安静区域。
2. 降低语言，给简单选择：“出去休息，还是戴耳机？”
3. 不在尖叫高峰时购买原本拒绝的物品。
4. 平静后再回到短时间任务。

**提前预防**：
- 出门前说明目的和时长
- 带等待包、耳机、安抚物
- 从短时间公共场所练习开始

**替代技能**：
- “我要休息”
- “太吵了”
- “我想走”

**强化策略**：
- 强化进店、等待、按清单完成、请求休息

**记录指标**：
- 哪个场所最难
- 可坚持时间
- 有效安抚物

**转介信号**：
- 频繁危险逃跑或严重攻击

## SCENE_011 危险逃跑、冲出门

**适用情况**：孩子突然跑向马路、冲出家门、在公共场所脱离家长。

**先排除**：
- 是否被某个强烈兴趣物吸引
- 是否想逃离噪音/人群/任务

**可能功能**：
- 获得想看的东西
- 逃离不舒服环境
- 追求奔跑感官刺激

**家长当下做法**：
1. 立即保证安全，必要时使用门锁、牵引绳等安全工具。
2. 不把追逐变成游戏。
3. 抓住后简短说明：“停，危险。”
4. 回到安全处教“停”“等我”“牵手”。

**提前预防**：
- 练习听到“停”立即停
- 出门前说清规则
- 先在安全封闭场地练习

**替代技能**：
- “我要去看”
- “我要跑”
- “牵手”

**强化策略**：
- 高强度强化停下、回头、牵手、等待

**记录指标**：
- 逃跑场景
- 逃跑方向/目标
- 是否听到“停”有反应

**转介信号**：
- 任何可能危及生命的逃跑都应尽快专业介入

## SCENE_012 感官过载：捂耳朵、尖叫、躲避

**适用情况**：孩子在噪声、灯光、人群、触觉刺激下崩溃。

**先排除**：
- 耳部疾病、疼痛、视觉问题
- 环境刺激是否明显过强

**可能功能**：
- 逃离感官刺激
- 自我保护
- 请求调节但不会表达

**家长当下做法**：
1. 先降低刺激，带到安静区域。
2. 不强迫继续暴露。
3. 用短句命名：“太吵了，我们休息。”
4. 提供耳机、帽子、安抚物。

**提前预防**：
- 预判高刺激环境
- 准备感官工具包
- 逐步脱敏，不突然强迫

**替代技能**：
- “太吵”
- “休息”
- “我要耳机”

**强化策略**：
- 强化提前表达不适
- 强化使用工具调节

**记录指标**：
- 触发刺激
- 可忍受时长
- 有效调节方式

**转介信号**：
- 严重影响出行、学习、进食或睡眠，建议OT/专业评估

## SCENE_013 只会重复话、不会功能性沟通

**适用情况**：孩子会背广告词、动画片台词，但不会用语言表达需求。

**先排除**：
- 听力和语言理解问题
- 是否有其他沟通方式

**可能功能**：
- 听觉自我刺激
- 表达情绪但形式不准确
- 不知道语言可以影响别人

**家长当下做法**：
1. 不急着纠正所有重复话。
2. 找到孩子当下真正想要什么。
3. 提供功能性句式：“我要车”“帮我打开”。
4. 孩子仿说功能句后立即满足合理需求。

**提前预防**：
- 每天制造自然沟通机会
- 把喜欢物放在看得见但够不到处
- 减少只背诵不互动的长时间屏幕

**替代技能**：
- 提要求
- 拒绝
- 请求帮助
- 请求继续/停止

**强化策略**：
- 强化任何有沟通意图的表达
- 从单词、手势、图片都可以开始

**记录指标**：
- 自发功能表达次数
- 重复话出现情境
- 可仿说的功能句

**转介信号**：
- 语言长期无功能性使用，建议语言治疗/ABA评估

## SCENE_014 不会主动提要求

**适用情况**：孩子想要东西但只拉人、哭、看着物品，不会说或指。

**先排除**：
- 是否不知道物品名称
- 是否之前哭闹就能直接获得

**可能功能**：
- 获得物品
- 用低成本方式沟通

**家长当下做法**：
1. 等待2-5秒，不马上代劳。
2. 提示最容易成功的表达：指、给图片、说单词。
3. 表达后立即给物品。
4. 逐渐减少提示。

**提前预防**：
- 把高动机物品放在透明盒中
- 每天安排多次短练习

**替代技能**：
- 指物
- 递图片
- 说“要”“打开”“帮我”

**强化策略**：
- 强化表达本身，不要求一开始发音完美

**记录指标**：
- 独立提要求次数
- 需要哪类提示
- 最有效动机物

**转介信号**：
- 2岁后完全没有沟通意图或技能退步，建议评估

## SCENE_015 不会等待

**适用情况**：等待食物、排队、轮流时立刻哭闹或抢夺。

**先排除**：
- 等待要求是否过长
- 是否孩子不知道等待多久

**可能功能**：
- 立即获得物品
- 逃避等待的不确定感

**家长当下做法**：
1. 从非常短的等待开始，比如1-3秒。
2. 使用计时器或等待卡。
3. 等待成功立即给强化。
4. 逐渐增加等待时间。

**提前预防**：
- 在低压力场景练等待
- 先练喜欢但不是最执着的物品

**替代技能**：
- “等一下”
- “还有多久”
- 看计时器

**强化策略**：
- 强化安静等待、手放好、看计时器

**记录指标**：
- 最长成功等待时间
- 等待失败触发点

**转介信号**：
- 等待时总伴随攻击或自伤

## SCENE_016 拒绝学习任务

**适用情况**：一看到桌面任务、写字、配对、读书就跑开或哭。

**先排除**：
- 任务是否过难
- 是否时间太长
- 是否视觉/精细动作困难

**可能功能**：
- 逃避任务
- 获得休息
- 避免失败感

**家长当下做法**：
1. 降低任务难度，让孩子先成功。
2. 用“先做1个，后玩1分钟”。
3. 必要时给辅助，不让孩子反复失败。
4. 完成后立即强化。

**提前预防**：
- 任务短、清楚、可完成
- 新旧任务混合
- 开始前展示强化物或活动

**替代技能**：
- “休息”
- “帮我”
- “太难了”

**强化策略**：
- 强化坐下、尝试、请求帮助、完成一小步

**记录指标**：
- 可持续学习时间
- 独立完成比例
- 逃避发生在哪类任务

**转介信号**：
- 长期强烈拒绝学习，需专业评估任务难度和教学方法

## SCENE_017 过度依赖提示

**适用情况**：孩子只有家长说很多遍、指着、扶着才做，不能独立完成。

**先排除**：
- 目标是否真的学会
- 提示是否给得太快太多

**可能功能**：
- 等待成人提示
- 害怕出错
- 习惯被代劳

**家长当下做法**：
1. 给指令后等待3-5秒。
2. 从最少提示开始。
3. 成功后立即强化独立反应。
4. 有计划撤退提示。

**提前预防**：
- 写清提示层级
- 避免每次立刻说答案

**替代技能**：
- 自己检查视觉提示
- 请求“帮一下”

**强化策略**：
- 独立完成的强化要明显大于被提示完成

**记录指标**：
- 独立/提示完成比例
- 使用哪种提示

**转介信号**：
- 多个技能长期无法撤退提示，需调整教学方案

## SCENE_018 不会和同伴玩

**适用情况**：孩子只自己玩，不理同伴，或靠近同伴但方式不合适。

**先排除**：
- 是否玩具太少、规则太难
- 是否语言表达不足

**可能功能**：
- 不理解社交规则
- 对同伴不可预测性感到焦虑
- 缺少进入游戏的技能

**家长当下做法**：
1. 选择结构清楚的短游戏。
2. 先教一句进入话术：“我可以玩吗？”
3. 成人辅助轮流。
4. 成功互动后立即表扬。

**提前预防**：
- 先和成人练，再和同伴练
- 从平行游戏到轮流游戏

**替代技能**：
- 请求加入
- 轮流
- 模仿同伴动作
- 结束游戏说“再见”

**强化策略**：
- 强化看同伴、递物、等待、轮流、模仿

**记录指标**：
- 主动靠近次数
- 互动持续时间
- 需要成人辅助程度

**转介信号**：
- 社交中频繁攻击或极度恐惧

## SCENE_019 过度沉迷手机/动画片

**适用情况**：不给手机就哭闹，长时间看屏幕，影响睡眠、语言和活动。

**先排除**：
- 是否缺少替代活动
- 是否屏幕是家长唯一安抚方式

**可能功能**：
- 获得强强化活动
- 逃避无聊、等待或任务

**家长当下做法**：
1. 不突然完全剥夺，先设清楚时长。
2. 用计时器提示结束。
3. 结束后立即提供替代活动。
4. 因哭闹延长屏幕时间要谨慎。

**提前预防**：
- 固定屏幕规则
- 准备高质量替代活动
- 睡前减少屏幕

**替代技能**：
- 请求“再看一分钟”
- 选择替代玩具
- 请求一起玩

**强化策略**：
- 强化按计时器结束、交还设备、转入替代活动

**记录指标**：
- 每日屏幕总时长
- 结束时哭闹强度
- 成功替代活动

**转介信号**：
- 屏幕剥夺导致严重攻击、自伤或睡眠严重紊乱

## SCENE_020 家长情绪失控、快要撑不住

**适用情况**：家长感到崩溃、无助、愤怒，担心自己会吼叫或打孩子。

**先排除**：
- 是否存在立即安全风险
- 是否家长或孩子有自伤/伤人风险

**可能功能**：
- 家长长期高压下资源耗竭
- 缺少支持和可执行方案

**家长当下做法**：
1. 先把孩子放到安全位置。
2. 家长短暂离开现场深呼吸或求助家人。
3. 不在情绪高峰时教育孩子。
4. 事后记录最难的触发点，寻求支持。

**提前预防**：
- 建立家长休息计划
- 减少同一时间处理多个目标
- 把目标改成小步成功

**替代技能**：
- 家长练习说：“我需要暂停一下。”
- 孩子练习“休息”“帮我”

**强化策略**：
- 家长也要记录小成功
- 强化自己使用了暂停而不是惩罚

**记录指标**：
- 家长压力最高的场景
- 可获得的支持资源
- 哪些策略能让局面缓和

**转介信号**：
- 家长担心自己会伤害孩子
- 家长出现自伤、自杀想法
- 家庭安全无法保障时立即寻求专业帮助或紧急援助

## 附录：回答优先级规则

当用户描述具体场景时，AI应优先：

1. 先确认安全和医学排除项
2. 解释可能行为功能，但不做诊断
3. 给当下处理步骤
4. 给下次预防方法
5. 教替代技能
6. 给记录指标
7. 明确转介信号

禁止：

- 推荐体罚、羞辱、恐吓
- 对自伤、攻击、逃跑等高风险场景只给普通建议
- 承诺“保证有效”
- 替代医生、BCBA或治疗师做个别化诊断
