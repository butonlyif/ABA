#### 活动1：点心时间提要求

**简介**：
利用孩子想吃零食的动机，教孩子用语言/手势提要求。

**适合年龄**：1-4岁

**目标技能**：
- 提要求（Mand）
- 仿说
- 手势沟通
- 语言表达

**所需材料**：
- 孩子喜欢的零食（饼干、果干、小馒头等）
- 透明罐子或容器

**具体步骤**：

```
步骤1：准备工作
- 选择孩子饥饿的时候（点心前）
- 选择孩子非常喜欢的零食
- 把零食放在透明但不容易打开的罐子里

步骤2：诱发动机
- 让孩子看到你拿着零食罐
- 摇晃发出声音吸引注意
- 把它放在孩子能看到但够不到的地方

步骤3：等待表达
- 等待孩子主动表达想要
- 如果孩子指、看向零食
- 你说"你要什么？"

步骤4：提示（如果需要）
- 如果孩子不表达
- 提示"说'我要饼干'"或"指一指"

步骤5：强化
- 孩子表达后立即把零食给他
- 大大表扬："你说了'我要饼干'，太棒了！"

步骤6：重复
- 吃完后继续练习
- 逐渐减少提示
- 强化独立表达
```

**变体和扩展**：

```
变体1：使用不同强化物
- 换不同的零食
- 换玩具、活动

变体2：使用不同表达方式
- 从手势→仿说→自发语言
- 逐渐增加语言复杂度

变体3：创造更多机会
- 让孩子帮忙拿罐子（够不到）
- 把零食放在高处
- 把零食装在不透明的盒子
```

**注意事项**：
```
✓ 选择孩子真的想吃/想要的东西
✓ 保持孩子饥饿状态（不要刚吃完饭）
✓ 立即给强化物
✓ 一开始可以给更多辅助，逐渐撤除
✓ 保持活动简短有趣
✓ 孩子不想参与时就暂停
```

#### 活动2：泡泡乐

**简介**：
吹泡泡是孩子喜欢的活动，利用这个动机教语言和社交。

**适合年龄**：1-5岁

**目标技能**：
- 提要求
- 目光接触
- 仿说（泡泡、吹）
- 社交互动
- 共同关注

**所需材料**：
- 泡泡水
- 泡泡棒
- 吹泡泡的嘴

**具体步骤**：

```
步骤1：建立动机
- 拿出泡泡水
- 开始吹泡泡
- 让孩子追泡泡、玩泡泡

步骤2：停止制造机会
- 盖上泡泡瓶
- 等待孩子表达想要继续

步骤3：等待和提示
- 如果孩子看向你
- 等待几秒看是否主动说
- 如果不说，提示"吹泡泡"或"还要"

步骤4：强化目光接触
- 每次吹之前
- 等孩子看你再吹
- 说"看！泡泡！"

步骤5：泛化
- 换不同大小的泡泡棒
- 在不同地方吹（室内/室外）
- 教"泡泡"、"吹"、"还要"
```

**变体和扩展**：

```
变体1：教不同的语言
- "泡泡"
- "吹"
- "还要"
- "大泡泡/小泡泡"
- "看泡泡"

变体2：社交扩展
- 教孩子吹给其他人看
- 教"给你吹泡泡"
- 和其他小朋友一起玩

变体3：加入动作
- 边吹边数
- 跳着追泡泡
- 用手接泡泡
```

**注意事项**：
```
✓ 选择孩子喜欢的泡泡
✓ 不要在有风的地方玩
✓ 泡泡水注意安全，不要入口
✓ 孩子追泡泡时注意安全
✓ 保持简短多次
✓ 每次吹之前等孩子看你
```

#### 活动3：音乐暂停游戏

**简介**：
播放孩子喜欢的音乐，在暂停时教等待和社交。

**适合年龄**：2-6岁

**目标技能**：
- 等待
- 目光接触
- 共同关注
- 社交互动
- 轮流

**所需材料**：
- 孩子喜欢的音乐/CD/手机
- 音响设备

**具体步骤**：

```
步骤1：建立动机
- 播放孩子喜欢的音乐
- 和孩子一起听/跳舞
- 享受音乐时光

步骤2：建立暂停规则
- 告诉孩子"音乐停了要拍手"
- 或"音乐停了要看妈妈"

步骤3：实施暂停
- 突然暂停音乐
- 等待孩子的反应

步骤4：强化正确反应
- 如果孩子看你/拍手
- 立即说"你做到了！"
- 立即播放音乐

步骤5：变化扩展
- 音乐停了要抱抱
- 音乐停了要跳舞
- 音乐停了要指耳朵

步骤6：加入轮流
- 音乐停了你做动作
- 音乐停了他做动作
- 轮流进行
```

**变体和扩展**：

```
变体1：加入等待
- 音乐停了等一下再播
- 逐渐延长等待时间

变体2：加入语言
- 教说"停"
- 教说"还要"
- 教说"音乐"

变体3：加入更多动作
- 跳
- 拍手
- 举手
- 转圈
```

**注意事项**：
```
✓ 选择孩子真的很喜欢的音乐
✓ 保持氛围愉快
✓ 不要突然太大声吓到孩子
✓ 等待时间从短开始
✓ 强化要即时
✓ 可以用沙锤等乐器增加变化
```

#### 活动4：厨房帮手

**简介**：
让孩子参与简单的厨房活动，在过程中教语言和技能。

**适合年龄**：2-6岁

**目标技能**：
- 命名（食材名称）
- 动词（洗、切、搅、倒）
- 指令服从
- 共同工作
- 生活技能

**所需材料**：
- 安全的儿童厨具
- 简单的食材
- 围裙

**具体步骤**：

```
步骤1：选择任务
- 洗菜（简单安全）
- 搅拌
- 倒材料
- 摆放餐具

步骤2：教任务语言
- 指着食材说名称
- 示范动作词
- "这是西红柿"
- "妈妈在切菜"

步骤3：让孩子参与
- "帮妈妈洗西红柿"
- 提供适合的辅助
- 确保安全

步骤4：教学机会
- "把西红柿给我"
- "洗得真干净"
- "下一个是什么？"

步骤5：完成和享受
- 一起完成食物
- 一起享用
- "你帮忙做的，真好吃！"
```

**变体和扩展**：

```
变体1：做简单食物
- 水果沙拉
- 三明治
- 饼干（简单步骤）
- 果汁

变体2：更多语言机会
- 描述味道
- 描述颜色
- 描述动作
- "洗洗洗"

变体3：家庭成员参与
- 叫其他人来吃
- 教"给奶奶吃"
- 分享食物
```

**注意事项**：
```
✓ 安全第一，远离热源和刀具
✓ 选择适合孩子能力的任务
✓ 不要期待完美
✓ 准备好清洁工具
✓ 保持耐心，允许弄乱
✓ 让孩子享受过程
```

#### 活动5：镜子游戏

**简介**：
利用镜子教孩子关注他人表情和自我意识。

**适合年龄**：1-4岁

**目标技能**：
- 目光接触
- 情绪识别
- 模仿
- 社交参照
- 自我意识

**所需材料**：
- 大镜子（落地镜或墙壁镜）
- 贴纸/道具（可选）

**具体步骤**：

```
步骤1：建立兴趣
- 带孩子到镜子前
- 让他看镜子里的自己
- 指指他、指指镜子里的"人"

步骤2：表情游戏
- 你对着镜子做表情
- "看妈妈的脸"
- 夸张地笑、皱眉、做鬼脸
- 观察孩子的反应

步骤3：模仿练习
- "和妈妈一样"
- 你做表情，等孩子模仿
- 立即强化

步骤4：镜子里的互动
- 你在镜子前挥手
- "看，这是谁？"
- "你好！"
- 教对着镜子里的自己招手

步骤5：扩展活动
- 贴贴纸（孩子喜欢的）
- 放在镜子上
- "妈妈/宝宝的镜子上有贴纸"
```

**变体和扩展**：

```
变体1：身体部位
- 指妈妈的脸
- 指镜子里的脸
- "这是脸"
- "脸在哪里？"

变体2：更多表情
- 开心（😊）
- 难过（😢）
- 生气（😠）
- 惊讶（😮）

变体3：装饰镜子
- 用可擦笔画
- 贴贴纸
- 挂照片
```

**注意事项**：
```
✓ 有些孩子可能不喜欢镜子，慢慢来
✓ 确保镜子安全（不易破碎）
✓ 可以用手机的前置摄像头作为替代
✓ 配合表情图片使用
✓ 每次时间不要太长
✓ 保持游戏氛围
```

**简介**：
利用孩子喜欢的儿歌，教语言、模仿和社交。

**适合年龄**：1-5岁

- 仿说（歌词）
- 动作模仿

```
推荐儿歌和游戏：

"两只老虎跑得快"——跑的动作

"五只小小鸟"——举起五指
"飞呀飞呀飞"——扇动手臂
```

```
- 唱的时候等待
- 看孩子是否主动参与

步骤5：泛化
- 换不同儿歌
- 在不同场合唱
- 配合其他孩子一起唱
```

**注意事项**：
```
✓ 选择适合孩子发展水平的
✓ 不要太长
✓ 夸张的表情和动作
✓ 配合孩子的兴趣
✓ 重复很重要
```

#### 活动8：穿衣练习

**简介**：
利用日常穿衣时间教自理和语言。

**适合年龄**：2-5岁

**目标技能**：
- 自理技能
- 命名（衣物名称）
- 方位词（上、下、里、外）
- 指令服从
- 轮流向

**所需材料**：
- 孩子的衣服（简单款式）
- 挂钩/衣架
- 穿衣辅助工具（如有必要）

**具体步骤**：

```
步骤1：建立常规
- 每天固定的穿衣时间
- 固定的顺序

步骤2：教选择
- "你想穿哪件？"
- 展示2-3件衣服
- 让孩子指/选

步骤3：分解任务
以穿T恤为例：
1. 找到领口
2. 找到前面（标签在哪）
3. 把头伸进去
4. 一只手穿进去
5. 另一只手穿进去
6. 拉下来

步骤4：口语机会
- "这是T恤"
- "你的手在哪里？"
- "拉下来"
- "穿好了！"

步骤5：增加独立性
- 逐渐减少辅助
- 孩子做更多步骤
- 分解任务逐步完成
```

**变体和扩展**：

```
变体1：使用视觉提示
- 照片展示穿衣顺序
- 视觉步骤卡

变体2：不同衣物
- 袜子
- 裤子
- 外套
- 鞋子

变体3：教"里外"概念
- 衣服里面/外面
- 袜子里面/外面
```

**注意事项**：
```
✓ 留足够的时间
✓ 不要着急
✓ 分解任务，一步步来
✓ 用简单款式的衣服开始
✓ 避免复杂的扣子、拉链
✓ 完成就大大表扬
✓ 保持耐心
```

#### 活动22：拼图

**简介**：
适合孩子能力的拼图游戏，练习问题解决和视觉能力。

**适合年龄**：2-6岁

**目标技能**：
- 视觉感知
- 问题解决
- 形状识别
- 位置概念
- 耐心

**拼图选择**：

```
年龄1-2岁：
- 木板嵌合拼图（1-5片）
- 形状配对板

年龄2-3岁：
- 2-4片拼图
- 大块拼图

年龄3-4岁：
- 6-12片拼图
- 复杂一点的图案

年龄4-6岁：
- 12-20+片拼图
- 更复杂的场景
```

**具体步骤**：

```
步骤1：选择合适难度
- 孩子能做成功的难度
- 不要太难

步骤2：示范
- 展示完整图
- "这是什么？"
- 演示拼的过程

步骤3：分解
- 一起拼
- "这块应该在哪里？"
- 提示位置

步骤4：辅助
- 握着孩子的手
- 一起拼
- 逐渐放手

步骤5：独立
- 让孩子独立完成
- 只在需要时帮助
- 完成后表扬

步骤6：进阶
- 增加片数
- 减少提示
```

**注意事项**：
```
✓ 选择适合难度的
✓ 失败了就降低难度
✓ 不要替孩子完成
✓ 完成就大大表扬
✓ 保持拼图有趣
✓ 可以分开几次完成
```
